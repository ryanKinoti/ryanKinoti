## ⚙️ DevOps Journey
 
> *"Shipping code is only half the job."*
 
My infrastructure path started with a Vercel deployment and one question that kept nagging at me: *what actually is a VPS?* That question has since opened up into a full thread I'm pulling on:
 
- **Self-hosted PaaS** on Hetzner via Dokploy and Coolify — real deployments, not tutorials
- Network configuration, firewall rules, and application-level security practices
- Metrics integration and observability fundamentals — learning to watch what's actually happening in production
- Security engineering as a design lens, not an afterthought
- **Next: Terraform** — managing 4+ production systems from code isn't optional, it's the only approach that scales without breaking you
